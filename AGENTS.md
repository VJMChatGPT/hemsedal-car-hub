# Dalmotorer

## Context del projecte
Aquesta és una app web de lloguer de cotxes per Dal Motorer.

## Com m'has de parlar
- Explica les coses de manera simple
- Assumeix que estic aprenent des de zero
- Abans de fer canvis, explica què tocaràs i per què

## Regles
- No barregis implementacions diferents en una sola tasca
- No toquis fitxers no relacionats
- No canviïs l'estil visual si no t'ho demano
- Si treballes el login admin, no toquis l'estructura de Supabase
- Si proposes canvis de Supabase, primer explica l'impacte

## Quan facis canvis
Sempre indica:
1. quins fitxers tocaràs
2. què canviaràs
3. com ho provaré
4. quins riscos hi ha
